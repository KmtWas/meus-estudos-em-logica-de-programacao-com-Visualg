
# Meus estudos em lógica de programação

Este repositório guarda meus primeiros exercícios de lógica de programação.
Estou começando na programação do zero, usando Visualg para praticar
entrada de dados, processamento e saída.

## Sobre este repositório

- `Primeiro Algoritimo- Olá, Mundo.alg`: primeiro contato com Visualg.
- `Segundo Algo- meu nome.alg`: leitura de dados e exibição de mensagem.
- `Valores Inteiros.alg`: exercícios com entrada de inteiros e operações simples.

Vou adicionando novos arquivos conforme avanço nas aulas e reformulando
os exercícios à medida que eu aprender coisas novas.
